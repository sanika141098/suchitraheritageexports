# Suchitra Heritage Exports

A premium wellness brand website showcasing Suchitra Heritage Exports with two specialized brands:

- **Suchitra Mindful Living** - Premium wellness services (mindfulness coaching, yoga, corporate wellness)
- **Suchitra Root & Ritual** - Premium Ayurvedic herbs and wellness products

## Features

- 🎨 Beautiful, responsive design with heritage aesthetic
- 🌿 Modern wellness branding
- 📱 Mobile-friendly interface
- ⚡ Fast and optimized performance
- 🎭 Comprehensive brand guide page

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Utility-first CSS framework
- **Wouter** - Lightweight routing
- **Lucide React** - Icon library

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Design System

### Colors

- **Forest Green** (#1B3A2E) - Primary brand color
- **Bronze** (#B8864B) - Heritage and luxury accent
- **Sand** (#E8DECD) - Warm neutral
- **Off-White** (#FAF8F5) - Clean background

### Typography

- **Playfair Display** - Serif font for headings (classical elegance)
- **Lato** - Sans-serif font for body text (modern clarity)

## Pages

- **Home** (`/`) - Main landing page with brand overview
- **Brand Guide** (`/brand-guide`) - Comprehensive visual identity system
